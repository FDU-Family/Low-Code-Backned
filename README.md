# Low-Code-Backned
base on GPT, the smater low-code platform

建立虚拟环境
```
python -m venv venv
```
运行虚拟环境
```
# cmd
.\venv\Scripts\activate

# powershall
& ./venv/Scripts/Activate.ps1

# bash
source ./venv/Scripts/activate
```

生成reqirement.txt
```
pip freeze > reqirement.txt
```

安装依赖
```
pip install -r reqirement.txt
```
